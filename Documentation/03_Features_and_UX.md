# Features & User Experience

> **Surfaces:** Features below are implemented on **Tier A** (iOS, Android, Mac, Windows). **Web-Public** does not host these modules unless explicitly called out as a public URL flow (e.g. invite landing).

## 1. Core Modules
- **Meeting Schedules:** Assign, view, and track meeting parts, public talks, and congregation duties.
- **Field Service Reports:** Seamless collection of monthly reports directly from the publishers' devices.
- **Publisher Records:** A secure, centralized database of publisher information, spiritual milestones, and historical activity reports.
- **Territory Management:** Digital assignment, tracking, mapping, and logging of congregation territories.

## 2. User Experience Highlights
- **Offline-First Capabilities:** Essential for the field ministry. The mobile apps will function without an internet connection, allowing users to view territories or log notes offline. The app will securely cache this data and automatically sync to the cloud once a connection is re-established.
- **Automated Notifications:** The system will send automated push notifications and/or emails to remind publishers of their upcoming meeting parts and prompt them to turn in their end-of-month reports.

## 3. Data Migration & Onboarding
- **Frictionless Data Import:** Built-in tools to easily migrate data from existing software (like NW Scheduler or Hourglass) via file imports, heavily reducing the friction for new congregations to adopt the platform.
- **Manual Entry:** Support for adding publishers and building out the congregation roster manually if a congregation prefers starting fresh.
